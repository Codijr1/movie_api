Documentation

This is a description and outline for the API

GET /all-movies Returns a list of all movies
GET /search-by-title Returns data on a specific movie by title
GET /genres Returns data about a genre
GET /directors Returns data about a director by name
PUT /register Allows a user to register
POST /update-username Allows user to update their username
PUT /my-list-add Allows user to add a movie to their list of favorites
DELETE /my-list-delete Allows user to remove a movie from their list of favorites
DELETE /user-controls Allows user to deregister
